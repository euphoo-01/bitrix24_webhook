# Приложение для получения компаний из Битрикс24

Это full-stack приложение, которое использует бэкенд на Node.js (Express) для получения списка компаний из Битрикс24 через REST API (вебхук) и отображает их на фронтенде, написанном на Vue 3.

## Структура проекта

Проект разделен на две основные части: клиентскую и серверную.

```bash
- /
  - client/             # Фронтенд на Vue 3
     - public/
      - src/
  - server/             # Бэкенд на Express.js
      - src/
          - controllers/
          - entities/
          - services/
      -  .env.example   # Пример файла для переменных окружения
  - package.json        # Корневой package.json для управления обоими приложениями
```

## Установка и запуск

**Важно!** Перед тем как начать, убедитесь, что в вашей системе установлен `Node.js` и пакетный менеджер `npm` при помощи команд в терминале:

```bash
node -v
npm -v
```

Для запуска проекта выполните следующие шаги:

1.  **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/euphoo-01/bitrix24_webhook.git
    cd bitrix24_webhook
    ```

2.  **Вставьте свой вебхук Битрикс24:**

    - Перейдите в директорию `server/`.
    - Создайте копию файла `.env.example` и назовите ее `.env`.
    - Откройте файл `.env` и вставьте ваш URL **входящего** веб-хука из Битрикс24:
      ```
      BITRIX_WEBHOOK=https://123.bitrix24.ru/rest/1/xxxxxxxxxxxxxxxx/
      ```

3.  **Установите зависимости:**

    - Вернитесь в корневую директорию проекта.
    - Выполните команду `npm install`.

4.  **Запустите проект в режиме разработки:**

    - Находясь в корневой директории, выполните команду:
      ```bash
      npm run dev
      ```

5.  **Откройте приложение в браузере:**
    - Перейдите по адресу `http://localhost:5173`, чтобы увидеть интерфейс.
